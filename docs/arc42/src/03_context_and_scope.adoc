== Context and Scope

=== Business Context

Magrathea ObjectStore provides an S3-compatible public object API. Users interact with the S3 API via HTTP to store and retrieve objects in buckets. Operators and the Admin UI use a separate internal `/admin/**` JSON API for storage policy, device, disk-set/topology, and configuration-as-code catalog visibility; this admin surface is not part of S3 API coverage.

.System Context — Magrathea ObjectStore
image::../../c4/images/SystemContext.png[System Context, width=600]

The system context shows:
- **User**: end user interacting via HTTPS — S3-compatible object API
- **Operator/Admin UI**: internal actor using `/admin/**` JSON endpoints and static documentation/UI assets
- **Magrathea ObjectStore**: Spring Boot 4 reactive object storage runtime with S3 and admin adapters
