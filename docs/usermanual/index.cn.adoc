== 前端概述

Magrathea ObjectStore 包含一个基于 Vue.js 的管理前端，运行在 8081 端口。

=== 访问前端

在浏览器中打开：

* `http://localhost:8081/` — 默认
* `http://<host>:8081/` — 从局域网的其他机器

前端由管理服务器（独立的嵌入式 Netty，端口 8081）提供，独立于端口 8080 上的 S3 API。

=== 页面

| 页面 | URL | 描述 |
|------|-----|------|
| 仪表盘 | `/` | 玻璃拟态设计概览 |
| 存储桶 | `/buckets` | S3 存储桶管理 |
| 存储策略 | `/storage-policies` | 存储策略配置（管理 API） |
| 文档 | `/docs` | 本用户手册 |

=== 语言支持

前端通过 vue-i18n 支持 5 种语言：

* English (en)
* Italiano (it)
* Español (es)
* Deutsch (de)
* 中文 (cn)

可以通过右上角的语言选择器切换语言。

=== 主题

界面采用玻璃拟态设计：

* 磨砂玻璃背景 (`backdrop-filter: blur`)
* 渐变覆盖层
* 平滑过渡
* 深色主题与强调色

=== 管理 API

管理 API 采用 REST 风格并支持 HATEOAS，在端口 8081 上可用：

* `GET /admin/health` — 健康检查
* `GET /admin/storage-policies` — 列出所有存储策略
* `GET /admin/storage-policies/{id}` — 获取单个策略
* `POST /admin/storage-policies` — 创建策略
* `PUT /admin/storage-policies/{id}` — 更新策略
* `DELETE /admin/storage-policies/{id}` — 删除策略

所有响应均包含用于 HATEOAS 导航的 `_links`。
