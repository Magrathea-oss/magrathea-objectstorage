== Frontend Overview

Magrathea ObjectStore includes a Vue.js-based administration frontend served on port 8081.

=== Accessing the Frontend

Open your browser to:

* `http://localhost:8081/` — default
* `http://<host>:8081/` — from another machine on the LAN

The frontend is served by the admin server (separate embedded Netty on port 8081), independent from the S3 API on port 8080.

=== Pages

| Page | URL | Description |
|------|-----|-------------|
| Dashboard | `/` | Overview with glassmorphism design |
| Buckets | `/buckets` | S3 bucket management |
| Storage Policies | `/storage-policies` | Storage policy configuration (Admin API) |
| Documentation | `/docs` | This user manual |

=== Language Support

The frontend supports 5 languages via vue-i18n:

* English (en)
* Italian (it)
* Español (es)
* Deutsch (de)
* 中文 (cn)

Language can be switched via the language selector in the top-right corner.

=== Theme

The UI uses a glassmorphism design with:

* Frosted glass backgrounds (`backdrop-filter: blur`)
* Gradient overlays
* Smooth transitions
* Dark theme with accent colors

=== Admin API

The admin API is REST with HATEOAS, available on port 8081:

* `GET /admin/health` — health check
* `GET /admin/storage-policies` — list all storage policies
* `GET /admin/storage-policies/{id}` — get a single policy
* `POST /admin/storage-policies` — create a policy
* `PUT /admin/storage-policies/{id}` — update a policy
* `DELETE /admin/storage-policies/{id}` — delete a policy

All responses include `_links` for HATEOAS navigation.
